      • getDefaultDisplayRotation

        @NonNull() Long getDefaultDisplayRotation()

        Gets default capture rotation for CameraX {@code UseCase}s.

        The default capture rotation for CameraX is the rotation of default {@code Display} at thetime that a {@code UseCase} is bound, but the default {@code Display} does not change in thisplugin, so this value is {@code Display}-agnostic.

        Seehttps://developer.android.com/reference/androidx/camera/core/ImageCapture#setTargetRotation(int)for instance for more information on how this default value is used.
